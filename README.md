E-Placer React Final

This project is a Vite + React starter with Tailwind.
Features:
- Public assets for images (/public/placements and /public/logos)
- Login (LocalStorage), Protected routes
- Apply flow with LocalStorage persistence
- My Applications page with cancel and status
- Load more tiles and smaller Airbnb-like cards

To run:
1. unzip into a folder
2. cd eplacer-react-final
3. npm install
4. npm run dev

Notes:
- No signup page (login accepts any email)
- Data stored in localStorage under keys: eplacer_apps, eplacer_slotAdjust, eplacer_user
